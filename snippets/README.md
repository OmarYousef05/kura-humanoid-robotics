# Portfolio-safe snippets

This folder is reserved for small, approved examples that illustrate general robotics engineering concepts from my experience without reproducing the KURA codebase.

Suitable future content may include:

- Original pseudocode for a high-level behaviour concept
- Condensed examples written specifically for this portfolio
- Generic ROS patterns that contain no team-specific implementation
- Small testing or debugging examples with synthetic data

Do not add:

- Full or partial private team source files
- Complete behaviour trees or internal algorithms
- Robot configuration, calibration values, credentials, or network details
- Internal logs, datasets, restricted research, or unapproved code
- Material copied from another team member's repository

Every addition should be reviewed for ownership, permission, and accidental disclosure. Examples must be clearly labelled as conceptual or portfolio-specific when they do not reproduce the deployed system.
